Overview Of Certificate Course In Cybersecurity Essentials: The Foundation

This Foundation Course on Cybersecurity introduces essential concepts and key terminologies in digital security. You’ll learn about network security, threat analysis, encryption, and ethical hacking, building a strong understanding of how to protect digital assets.By the end of the course, you will have a solid grasp of the fundamental principles of cybersecurity, key terminologies and be ready to explore more advanced areas of digital security.

This 10 weeks foundational module includes

Module 1- What is Cyber security?

  • Importance of cyber security in today’s world.

  • Understanding the cyber space

  • Difference between cybersecurity and Information security

Module 2- Network security

  • History of the Internet

  • Types of network

  • Network protocol

  • Network Intrusion

  • Network security measures

Module 3: Application security

  • Introduction to Application Security
  • Importance of Application Security
  • Secure Software Development Lifecycle (SDLC)
  • Threat Modeling
  • Secure Coding Practices
